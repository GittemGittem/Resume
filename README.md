This project is a collection of showcases, to display my technical skill, and knowledge of computer technology; a resume. My goal with this repository is to make it easy for a potential employer or someone interested in my skills to find the qualities they are looking for in me as an employee.

Repository Structure:
    | Showcases:
        | Category:
            A directory containing information and examples which display my knowledge in a CIT category
            - sources.txt:
                a text file which contains links to my github repositorities related to the category
            - experience.txt:
                a text file which describes my experiences revolving around a category
        
I'm not the most experienced in every category, but hopefully I should be able to display my wide experience with computers and technical concepts, as well as my eagerness to learn more
My greatest asset is that I have a wide knowledge base, that can easily be expanded upon in order to fit new roles.

if you can't find what you are looking for on here, please let me know so I can add it, or develop the required skill if I lack it.

I want to know what YOU are looking for in a new hire, so any feedback is welcome.
